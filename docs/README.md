# Repository Structure

- `results`: Top-level folder which contains all viewable results. Organized by strategies and filters implemented.
    - HTML files contain full descriptions of trades, represented as dataframes, and related graphs.
      - Description is given in the name of the file
      - Further strategies related clarification and details will be given in future versions
    - JSON summary of trading strategies and their related metrics is shown
    - List of available results/graphs
      - Regime based cross asset mv and momentum strategy implementation results
      - Cross asset mv with neutrality conditions 
      - Time based mv
